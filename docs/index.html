Zero-trust privacy
Your voice stays
on your hardware

No servers. No API keys. No analytics. Audio goes from microphone to mlx-whisper to your cursor — then it's gone. Nothing is ever written to disk.

🔒

No network requests

After the initial model download from HuggingFace, LocalWhisper never makes a single network call. Verify it yourself — it's open source.

💾

RAM-only processing

Audio is captured as float32 in memory, transcribed, and immediately discarded. Nothing is ever saved to disk — not even temporarily.

🔓

Fully open source

Every line of code is on GitHub. Audit the source, build it yourself, verify every claim. Transparency is not optional — it's the architecture.

Translation
Speak in one language,
type in another

Any-to-any translation powered by a local LLM (Llama 3.2 3B). For input-to-English, Whisper handles it natively — no extra model needed.

Deutsch
"Hallo, wie geht es dir?"
→
English
"Hello, how are you?"
🌐

12 languages

Deutsch, English, Français, Español, Italiano, Português, Nederlands, Polski, Japanese, Chinese, Korean, Russian.

🧠

Runs locally

The translation LLM (1.8 GB, 4-bit quantized) runs entirely on Apple Silicon via mlx-lm. No API calls, no cloud.

⚡

Auto-download

The translation model downloads automatically on first use. Toggle translation on/off in the Settings window, pick your languages, and start dictating.

Text processing
Clean up your speech,
keep your meaning

An on-device LLM (Llama 3.2 3B) post-processes transcriptions to fix grammar and adjust tone — without changing what you said. Works in any language.

Clean up

“so yeah i went to the uh store and bought like some stuff”
↓
“I went to the store and bought some stuff.”

Formal

“hey can you send me that report thing asap”
↓
“Could you please send me the report at your earliest convenience?”

Casual

“I would like to inform you that I shall be arriving at approximately 3pm”
↓
“Hey, I'll be there around 3!”
🧠

Meaning preserved

The LLM adjusts tone and fixes errors, but never adds or removes content. Your words, polished.

🌐

Any language

Text processing works in all 12 supported languages. Speak German, get polished German back.

⚡

Combine with translation

Use text processing and translation together — speak casually in German, get formal English output.

Get started

Download the app

Grab the .dmg, drag to Applications. On first launch go to System Settings → Privacy & Security → Open Anyway. Then grant Mic & Accessibility permissions.
